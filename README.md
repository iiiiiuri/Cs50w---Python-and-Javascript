# 🚀 Project 0 : Front-end para Google Search, Google Image Search e Google Advanced Search

<img src="markdown/banner.png" width="100%">


## 📝 Descrição

Este projeto é uma implementação de front-end para três funcionalidades do Google: Google Search, Google Image Search e Google Advanced Search. Utilizando formulários HTML e enviando dados via GET para os servidores do Google, criamos interfaces que permitem ao usuário realizar buscas de maneira semelhante às páginas originais do Google.

## 🏗️ Estrutura do Projeto

O projeto consiste em três páginas HTML:

- `index.html`: Página principal para Google Search.
- `imgSearch.html`: Página para Google Image Search.
- `advSearch.html`: Página para Google Advanced Search.

## 🎯 Funcionalidades

### Google Search (`index.html`)

- Campo de Busca: Campo de texto centralizado com bordas arredondadas para digitar a consulta.
- Botão Google Search: Botão centralizado abaixo do campo de busca.
- Botão I'm Feeling Lucky: Botão que redireciona o usuário diretamente ao primeiro resultado da busca.
- Links de Navegação: Links no canto superior direito para navegar para as páginas de Image Search e Advanced Search.

### Google Image Search (`imgSearch.html`)

- Campo de Busca: Campo de texto para digitar a consulta de imagens.
- Botão Image Search: Botão para realizar a busca de imagens.
- Link de Navegação: Link no canto superior direito para retornar à página principal de Google Search.

### Google Advanced Search (`advSearch.html`)

- Campos de Busca Avançada:
    - "Find pages with… all these words:"
    - "Find pages with… this exact word or phrase:"
    - "Find pages with… any of these words:"
    - "Find pages with… none of these words:"
- Botão Advanced Search: Botão azul com texto branco, semelhante ao do Google.
- Link de Navegação: Link no canto superior direito para retornar à página principal de Google Search.

## 🚀 Como Utilizar

- Acesse a Página Principal: Abra o arquivo `index.html` em um navegador.
- Realize uma Busca Simples:
    - Digite a consulta no campo de busca.
    - Clique em "Google Search" para ver os resultados.
    - Clique em "I'm Feeling Lucky" para ir diretamente ao primeiro resultado.
- Realize uma Busca de Imagens:
    - Navegue até `images.html` através do link no canto superior direito.
    - Digite a consulta de imagens e clique em "Image Search".
- Realize uma Busca Avançada:
    - Navegue até `advanced.html` através do link no canto superior direito.
    - Preencha os campos desejados e clique em "Advanced Search".

## 🎨 Estilo e Aparência

O CSS utilizado no projeto busca imitar a estética do Google, com campos de entrada e botões estilizados de forma simples e intuitiva.

## 📋 Requisitos

- Um navegador web atualizado para visualizar e interagir com as páginas HTML.

## 📌 Observações

- A funcionalidade "I'm Feeling Lucky" pode apresentar um aviso de redirecionamento devido a uma característica de segurança do Google.
- Este projeto é uma simulação de interface e utiliza os servidores do Google para exibir os resultados de busca.
